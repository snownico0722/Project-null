# Linux DO 溶解计划

一个运行在浏览器本地的 Linux DO 用户脚本。它可以把指定用户的可见身份替换为随机假名，减少头像、装扮和签名带来的干扰；同时支持隐藏相关主题、清洗帖子标题，并在发送回复时无感地把当前页面出现过的 `@` 假名映射回真实用户名。

> 本项目只改变你自己的浏览器显示和发帖前的编辑内容，不会修改 Linux DO 上的任何数据，也不会让其他用户看到你的设置。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 或其他兼容 UserScript 的浏览器扩展。
2. [点击安装 Linux DO 溶解计划](https://raw.githubusercontent.com/snownico0722/Project-null/main/linuxdo-Project-null.user.js)。
3. 在扩展页面确认安装，然后访问 [Linux DO](https://linux.do/) 即可使用。

脚本仅匹配 `https://linux.do/*` 和 `https://www.linux.do/*`。

## 使用

- 点击 Linux DO 顶部导航中的“溶解计划”按钮，打开设置面板。
- 在用户卡片或用户相关操作中选择“溶解此人”，即可将该用户加入本地列表。
- 可单独开关头像替换、身份装饰清除、签名隐藏、假名映射、主题隐藏和标题清洗；标题清洗只作用于已加入溶解名单的主题作者。
- 纯净模式默认关闭；开启后会溶解所有可见用户并清洗所有帖子标题，同时停用“隐藏其发布的帖子”。
- “立即重置随机身份”会重新生成当前设备上的随机身份映射；已保存的溶解用户列表仍会保留。
- 回复中输入当前页面出现过的 `@随机名` 时，发送会自动映射回真实用户名；普通 `@用户名` 不会被改写。输入 `\@随机名` 可保留为普通文字，输入 `@@用户名` 可绕过映射。

## 数据与隐私

配置、溶解用户列表和随机映射只保存在浏览器的 UserScript 存储中，不会上传到第三方服务。卸载脚本或清除扩展数据后，本地设置也会消失。

## 兼容性

脚本针对 Linux DO 当前的 Discourse 页面结构编写，并适配页面局部刷新和常见的单页路由切换。Linux DO 页面结构变化后，部分功能可能需要更新脚本版本。
